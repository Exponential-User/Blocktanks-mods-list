---Installation---

1. Open DevTools (F12 or Ctrl+Shift+I)

2. Go to the Console tab

3. Copy and paste the Join-Alert.js script into the console

4. Press Enter

5. The script will run and the alert will be displayed in the chat once someone joins the server
